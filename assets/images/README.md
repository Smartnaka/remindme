# Assets Required

This folder needs the following image files:

1. **icon.png** (1024x1024)
   - App icon for iOS and Android
   - Used in app.json

2. **splash-icon.png**
   - Icon shown on splash screen
   - Used in app.json

3. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon foreground
   - Used in app.json

4. **favicon.png** (48x48 or 32x32)
   - Web favicon
   - Used in app.json

## Quick Solution

You can temporarily use any PNG images with these names to get the app running. Replace them with proper icons later.

## Recommended Tools

- Use online icon generators
- Export from design tools (Figma, Sketch)
- Download free icons from Flaticon or Icons8
